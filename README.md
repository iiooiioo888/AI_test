### 🔊 音频保留 (v2.0)
所有特效和扩展模式现在支持**自动音频保留**：
- 特效处理：自动提取源音频并合并到输出
- 扩展模式：音频随视频同步处理（循环、倒放、慢放等）
- 无音频源：自动降级为纯视频输出

## 快速开始

```bash
# 安装依赖
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 启动
python app.py
```

访问 http://localhost:8888

## 技术栈

- **后端**: Python + FastAPI + Uvicorn
- **前端**: 原生 HTML/CSS/JS (暗色主题)
- **视频处理**: OpenCV + FFmpeg (管道式处理)
- **实时通信**: WebSocket 推送进度

## 支持格式

- 输入: MP4, AVI, MOV, MKV, WebM, GIF
- 输出: MP4 (H.264 + AAC)
