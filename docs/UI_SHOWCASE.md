### 本地部署步驟

```bash
# 1. 克隆代碼
git clone https://github.com/iiooiioo888/AI_test.git
cd AI_test

# 2. 安裝依賴
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 3. 啟動應用
python -m app.main

# 4. 訪問應用
# http://localhost:8888
```

## 🎨 UI 設計特點

### 配色方案

| 顏色 | 用途 | Hex |
|------|------|-----|
| **主背景** | 頁面背景 | `#0a0a0f` |
| **卡片背景** | 卡片容器 | `#1e1e2e` |
| **主色** | 強調/按鈕 | `#6366f1` |
| **成功** | 完成狀態 | `#10b981` |
| **警告** | 審核狀態 | `#f59e0b` |
| **錯誤** | 失敗狀態 | `#ef4444` |
| **文字** | 主要文字 | `#ffffff` |
| **次要文字** | 輔助文字 | `#a0a0b0` |

### 動畫效果

| 動畫 | 用途 | 時長 |
|------|------|------|
| **fadeIn** | 頁面切換 | 0.4s |
| **slideUp** | 模態框 | 0.3s |
| **pulse** | 狀態指示器 | 2s |
| **spin** | 加載器 | 1s |
| **shimmer** | 進度條 | 2s |

## 📸 截圖指南

### 使用 Chrome DevTools

1. 打開應用 `http://localhost:8888`
2. 按 `F12` 打開 DevTools
3. 按 `Ctrl+Shift+P` 打開命令面板
4. 輸入 `screenshot`
5. 選擇 `Capture full size screenshot`

### 使用 Firefox

1. 打開應用
2. 按 `F12` 打開開發者工具
3. 點擊 `...` → `截圖`
4. 選擇 `截取整個頁面`

### 使用擴展程序

推薦擴展：
- **GoFullPage** (Chrome)
- **Fireshot** (Firefox)
- **Nimbus Screenshot** (Chrome/Firefox)
